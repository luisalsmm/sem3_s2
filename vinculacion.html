Maneras para vincular HTML con CSS

Utilizando la etiqueta

Este tipo de vinculacion se refiere a utilizar la vinculacion dentro del body. Ejemplo usar una etiqueta style en una linea de codigo.

De manera inline como atributo del elemento

Este tipo de vinculacion se refiere a utilizar un estilo dentro de una etiqueta como atributo, por ejemplo agregarle a una etiqueta "p" un style="color".

De manera externa mediante un archivo con extension .css

Este tipo de vinculacion se refiere a utilizar un archivo con extension en este caso estamos importando del archivo style.css y en ese archivo es donde se define el estilo, esto lo hacemos para tener mayor orden en el codigo.
