4 Conceptos importantes

Selectores

Nos indica a que elementos HTML, se aplicaran los estilos.

Herencia

Esto refiere a algunos elementos que heredan propiedades, por ejemplo colocar un p dentro de un body, si aplicamos estilo al body, el elemento "p" hereda el estilo.

Cascada

Esto quiere decir que en caso de aplicar diferentes estilos a un selector, prevalece el ultimo aplicado.

Especificidad

cada selector tiene un Nivel con un valor de importancia. ejemplo.

NIVEL DE SELECTORES DE ESPECIFICIDAD
Etiqueta 1
clases 10
id 100
inline 1000
!important infinito
